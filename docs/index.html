Path of Exile 2 Build Planner

The ultimate free .build web tool for Path of Exile 2. Convert existing PoB links directly into in-game .build files, update your current guides, or experiment and create new build variants from scratch.

Launch Build Planner

Core Features

✨

Convert PoB to .build Format

Easily import builds from Path of Building (PoB) links (pobb.in, pastebin, etc.) and instantly convert them into the official Path of Exile 2 .build format for in-game use.

🌳

Interactive Passive Tree

Fully interactive PoE2 passive skill tree. Plot your route, allocate nodes, and calculate level requirements dynamically.

💎

Skill & Support Gems

Search and assign active skill gems and support gems. Build complex link setups to maximize your damage output.

🛡️

Equipment Database

Plan your gear across all inventory slots. Full integration with the latest unique items database to finalize your build.

💾

Import & Export

Save your progress locally or download the finalized `.build` file to load directly into the Path of Exile 2 game client.
